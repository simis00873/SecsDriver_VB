# SECS-Driver

This program uses vb.net to practice SEMI standard.

SEMI standard defines the format of the message to pass between the host and the equipment

## SEMI Standard

| Standard | Name |
|----------|------|
| E23 | Cassette Transfer Parallel I/O Interface |
| E84 | Reinforce E23 
| E4 | SECS-I |
| E37 | HSMS |
| E5 | SECS-II |
| E30 | GEM |
| E82 | IB-SEM |
| E88 | STK-SEM |
